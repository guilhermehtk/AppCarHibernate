Instruções para a execução do programa

1 - Importe o arquivo de dump no MySQL. 
2 - Verifique as configurações do MySQL no arquivo xml do Hibernate e no ConnectionFactory dos relatórios.
3 - Execute o programa pelo NetBeans, para gerar o .jar use a ferramenta da IDE e na pasta dist, verifique que contenha a pasta /relatorios e /src/relatorios/xml com os arquivos .jrxml e a logo para que os relatórios sejam gerados corretamente.
